# SVTC Admin App

Admin web application for SVTC staff.

## MVP Scope

- Admin login.
- Manage students and activation status.
- Import students by CSV.
- Manage courses and classes.
- Create lessons manually.
- Generate recurring weekly class schedules.
- Reschedule or cancel individual lessons.
- Send class notifications.
- View basic notification delivery status.
